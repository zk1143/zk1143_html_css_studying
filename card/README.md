# 學程式練習的卡片

10 款用純 CSS 實作的現代卡片特效，無 JavaScript。

## 預覽
直接打開 `card.html` 即可在瀏覽器看到所有效果  
（建議用 Chrome / Edge / Firefox 最新版）

## 卡片列表

1. 霓虹發光 (Neon Glow)  
2. 毛玻璃效果 (Glassmorphism)  
3. 流動漸層 (Gradient Flow)  
4. 3D 翻轉卡片 (Flip Card)  
5. 黑金奢華 (Imperial Dark Gold)  
6. 手繪風虛線邊框 (Sketch Style)  
7. 流雲毛玻璃 (Cloud Glass)  
8. 金屬浮雕 (Metallic)  
9. 純 CSS 煙火 (Fireworks)  
10. 旋轉星空 (Rotating Stars)
